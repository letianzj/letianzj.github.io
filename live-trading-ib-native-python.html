Posted on 2018-01-28 Edited on 2020-08-17 In Systematic Investment , Trading Ssytem Disqus:

Introduction

This post domonstrates how to use quanttrading2 to connect to Interactive Brokers and live trade. Quantrrading2 is pure Python and the brokerage API is also native Python so in total the solution is 100% Python. Interactive Brokers is a popular brokerage among quant traders thanks to its powerful and robust Application Programming Interface (API). A few years ago, I open sourced a trading system with connection to IB C# API. Now with IB's new Native Python API library, it is a good idea to build strategies in Python in order to leverage Python's machine learning toolkits.

Historical Data Download

With all the components in place, all you need to do is to write your own strategy class. An example is MADoubleCross. This simple strategy buys on golden cross when short-term moving average crosses long-term moving average from below, and sells on death cross when the former crosses the later from above.

To optimize for example the short-term and long-term parameters, toggle do_optimize flag to True. To keep quanttrading2 nimble, the parameter_search is also moved out of the package, because of its dependency on pyfolio.
